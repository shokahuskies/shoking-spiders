# shoking-spiders
software system for decentralized UAV

multi-agent-sim: communication simulator for decentralized multi-agents. Assigns Leaders, and Re-assings leaders if the OG leader is dead. 

Traj-planning: trajectory planning and collision avoidance engine using greedy and A* algorithms. 

EKF(in progress): extended kalman filter in C++ to deal with the sensor noise for the robot

Currently trying to fuse those together and make tweaks to the traj-planning
